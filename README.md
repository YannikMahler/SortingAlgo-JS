# Sorting Algorithm Visualization 

This simple project visualize using Bootstrap 5.1 + JavaScript.

### Implemented Algorithms:

- Bubblesort
- Quicksort
- Mergesort
- Insertionsort
- Selectionsort
- Radixsort (WiP)


[Check out the site ](https://yannikmahler.github.io/SortingAlgo-JS/)
